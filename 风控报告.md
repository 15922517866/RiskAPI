# 风控报告

@method

```
拉取风控系统中的风控报告
```

@introduction

```
拉取指定用户最新的风控报告
```

@URL

```
/
```

@request  method

```
http/post
```

@request  param

| 参数 | 必选 | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| orderId | Y | String | 借款订单ID |
| submitTime | Y | long | 本次提交的时间戳 |

@request  json

```js
{
    "orderId":"xxxxxxxxxxxxxx",
    "submitTime":xxxxxxxxxxx
}
```

@response  param

| 返回字段 | 字段类型 | 父类 | 说明 |
| :--- | :--- | :--- | :--- |
| code | int | 无 | 系统响应码：1000，其他你们定义 |
| message | String | 无 | 错误信息 |
| data | Object | 无 | 返回的JSON数据集合，详情见下文@@data  param |

@@data  param

| 参数 | 字段类型 | 说明 |
| :--- | :--- | :--- |
| checkName | String | 风控检查项 |
| memo | String | 比对值 |
| result | String | 比对结果 |

@response  json

```js
{
    "code":  xxx,
    "message":"xxx",
    "data":[
        {
            "checkName":"直接联系人中黑名单人数>=1",
            "memo":"2",
            "result":"Abnormal"
        }，
        {
            "checkName":"近6个月无通话天数>90天",
            "memo":"60",
            "result":"Normal"
        }，
        {
            "checkName":"进件LBS城市与居住地址不一致",
            "memo":"不一致",
            "result":"Review"
        },{
            ...
        }
    ]
}
```
